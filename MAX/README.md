## Website Overview
TECHMAX Tours & Safaris is a modern, responsive website designed to showcase and book various tourism services in Zanzibar, Tanzania. The site offers tours, safaris, car rentals, packages, excursions, and more, with a focus on providing an immersive user experience through interactive elements, galleries, and booking forms.

## Website Structure

### Main Pages
- **index.html**: Homepage featuring hero section, image slider, testimonials, service overview, and Airbnb rentals
- **about.html**: About the company and services
- **contact.html**: Contact form, information, and Google Maps integration
- **tour.html**: Tour listings with search/filter functionality and calendar integration
- **gallery.html**: Photo gallery of tours, events, excursions, and nature
- **blog.html**: Blog posts and articles
- **fag.html**: Frequently Asked Questions (FAQ)
- **policy.html**: Company policies and terms

### Service Pages
- **car-rental.html**: Car rental services and listings

## How to Run the Website

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Local web server (optional, but recommended for full functionality)

### Local Development Setup
1. **Clone/Download** the project files to your local machine
2. **Navigate** to the project directory
3. **Open** `index.html` in your web browser
4. For full functionality (PHP backend):
   - Set up a local server (XAMPP, WAMP, or similar)
   - Place files in the server's root directory
   - Ensure PHP and MySQL are running

## Navigation Instructions for Newcomers

### Getting Started
1. **Homepage (index.html)**: Start here to get an overview of services
2. **Explore Services**: Use the quick navigation cards to jump to specific sections
3. **Book a Tour**: Click "Book Now" or "Book a Tour" buttons throughout the site

### Main Navigation Flow
1. **Tours** → Search/Filter → View Details → Book
2. **Packages** → Select Package → View Details → Book
3. **Car Rental** → Browse Cars → View Details → Book
4. **Excursions** → Browse Activities → View Details → Book
5. **Gallery** → Browse Photos → Filter by category
6. **Blog** → Read Articles → Contact for inquiries
7. **Contact** → Fill form or use WhatsApp button

### Interactive Elements
- **Calendar Icon**: Fixed position, shows upcoming events
- **Video Short Icon**: Left side, links to video content
- **Scroll Up Button**: Bottom right, appears on scroll
- **WhatsApp Button**: Contact page, direct messaging
